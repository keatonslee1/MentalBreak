     Metrics Animation System Implementation Plan

     Overview

     Implement animated metrics changes with SFX, visual highlights, and sequential 
     triggering when multiple metrics change at once.

     Requirements Summary

     - SFX: Play up_swell.wav or down_wobble.wav when a metric changes
     - Highlight: Background color pulse on the metric panel
     - Animation: Bar grows/shrinks over 1 second with smooth interpolation
     - Sequencing: Multiple changes animate in order (Engagement → Sanity → Suspicion) 
     with 0.33s delay between starts
     - Skip Logic: Skip unchanged metrics; skip Suspicion if $suspicion_hud_active is 
     false
     - Queueing: New changes queue and play after current sequence finishes

     Architecture

     Create a new MetricsAnimator component that works alongside MetricsPanelUI:
     - MetricsPanelUI handles layout/creation (unchanged core behavior)
     - MetricsAnimator handles change detection, SFX, and animation
     - Separation of concerns = easier to maintain/disable

     Key Methods:
     - CheckForChanges() - Compare current vs previous Yarn variable values
     - QueueAnimations(List<MetricChange>) - Add changes in priority order
     - ProcessQueue() - Coroutine: process queue with 0.33s staggered starts
     - AnimateMetric(MetricChange) - Coroutine: 1s animation with Lerp
     - PulseHighlight(GameObject) - Coroutine: background color pulse
     - PlayMetricSFX(MetricType, bool isUp) - Play appropriate sound

     Change Detection Flow:
     1. Poll Yarn variables on interval (0.1s like MetricsPanelUI)
     2. Compare to previous values (skip if equal using Mathf.Approximately)
     3. On first load, set previous values without animating
     4. Build list of changes, sort by priority (Engagement=0, Sanity=1, Suspicion=2)
     5. Skip Suspicion if $suspicion_hud_active is false
     6. If not currently animating, start ProcessQueue() coroutine

     Animation Coroutine:
     IEnumerator AnimateMetric(MetricChange change)
     {
         // Play SFX at start
         PlayMetricSFX(change.Type, change.NewValue > change.OldValue);

         // Start highlight (runs parallel)
         StartCoroutine(PulseHighlight(panel));

         // Animate fill over 1 second
         float elapsed = 0f;
         while (elapsed < animationDuration)
         {
             elapsed += Time.deltaTime;
             float t = Mathf.SmoothStep(0, 1, elapsed / animationDuration);
             fillImage.fillAmount = Mathf.Lerp(oldFill, newFill, t);
             valueText.text = $"{Mathf.Lerp(oldVal, newVal, t):F0}%";
             yield return null;
         }
     }
